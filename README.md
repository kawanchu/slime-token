# Slime Token
Simple original token wallet

## Requirements
### Libraries
```
$ node --vsersion
v9.7.1

$ yarn --version
1.9.4

$ truffle --version
Truffle v4.1.14 - a development framework for Ethereum

$ ganache-cli --version
Ganache CLI v6.1.8 (ganache-core: 2.2.1)

# optional
$ vue --version
2.9.6
```

### Metamask
Install [Metamask browser extension](https://metamask.io/)

## Setup
### Deploy contract
```
$ yarn rpc
$ truffle compile
$ truffle migrate
```

## Run
```
$ yarn rpc
$ yarn dev
```
